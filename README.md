# Cinema Seats Picker 🎬

A cinematic movie seat booking experience built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

This project was inspired by modern ticketing platforms like Seats.io, Ticketbooth, and ShowClix. The goal was to create a seat booking flow that feels interactive, animated, and closer to a real cinema experience.

## Features

- Interactive cinema seating chart
- Hover tooltips for seat details
- Animated seat selection
- Persistent booked seats using localStorage
- Promo code support (`111` for 10% off)
- Checkout flow with order summary
- Upsell section for snacks and merch
- Animated confirmation screen
- Responsive layout
- GitHub Pages deployment support

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

Then open:

```txt
http://localhost:3000
```
